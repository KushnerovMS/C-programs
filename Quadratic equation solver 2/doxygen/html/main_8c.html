◆ getRootsOfQE()

int getRootsOfQE ( double  a,
double  b,
double  c,
double *  x1,
double *  x2 
)

Calculate roots of quadratic equation.

Аргументы
[in]a,b,ccoefficients of quadratic equation
[out]x1,x2roots of quadratic equation
Возвращает
count of roots

At first function calculate \( D \) - discriminant ( \( D = b^2 - 4\cdot a\cdot c\)).

Then if \( D < 0 \) equation haven't got roots;

If \( D < ( 2\cdot x\cdot \varepsilon)^2 \) \( \left( \text{that is we check} \dfrac{\sqrt{D}}{2\cdot a} < \varepsilon \right) \) it can get \( c = 0 \) and calculate \( x = - \dfrac{d}{2\cdot a} \);

If \( \left| a\cdot c^2 \right| < \varepsilon \cdot \left| b^3 \right| \) \( \left( \text{that is we check} \left| \dfrac{a\cdot c^2}{b^3} \right| < \varepsilon \right) \) it can get \( a = 0 \) and calculate \( x = - \dfrac{c}{b} \) which is easy to prove;

If \( b = 0 \) it is return if \( c = 0 \) 0 else _INFINITY

in other cases it calculate \( x_1 = \dfrac{- b - \sqrt D}{2\cdot a} \) , \( x_2 = \dfrac{- b + \sqrt D}{2\cdot a} \)

( \( \varepsilon = \) ROOT_PRECISION)
